✨ ЭФФЕКТЫ И АНИМАЦИИ (20+):

CRT Эффекты:
  1. Скан-линии (scanlines)
  2. Хроматическая аберрация (chromatic aberration)
  3. Вибрация (vibration/jitter)
  4. Bloom (свечение)
  5. CRT curve (кривизна экрана)
  6. Динамичный шум (noise pattern)

Глитч-анимации:
  7. Glitch transition (переход с глитчем)
  8. TV static flash (вспышка помех)
  9. TV noise (движущиеся помехи)
  10. Crash screen (ломаный экран)
  11. Distort (искажение clip-path)
  12. Dissolve (растворение)
  13. Glitch text (мерцающий текст)

Другие анимации:
  14. Product enter (вход товара с перспективой)
  15. Product exit (выход товара)
  16. Logo vibration (вибрация логотипа)
  17. Logo glow (динамичное свечение)
  18. Button shine (блик на кнопке)
  19. Danger pulse (красная пульсация)
  20. Media enter (появление медиа)

🚀 КАК ИСПОЛЬЗОВАТЬ:

1️⃣ Локальный запуск:
   Windows: run-local-server.bat
   Linux/Mac: ./run-local-server.sh
   Python: python -m http.server 8000
   
   Откройте: http://localhost:8000

2️⃣ Добавить товары:
   Редактируйте: /data/products.json
   Добавьте медиа: /assets/media/

3️⃣ Развернуть на GitHub:
   git init
   git add .
   git commit -m "Initial"
   git remote add origin ...
   git push -u origin main
   
   Settings → Pages → Deploy from branch (main)
